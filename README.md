## 비밀번호 생성기 및 강도 평가

이 도구는 랜덤 비밀번호를 생성하고 비밀번호의 강도를 평가합니다. 비밀번호의 강도는 길이, 대소문자, 숫자, 특수 문자를 기준으로 평가됩니다.

### 사용 방법:
1. 생성할 비밀번호의 개수를 입력합니다.
2. 원하는 비밀번호 길이를 입력합니다.
3. 프로그램은 비밀번호를 생성하고 다음과 같은 강도로 평가합니다:
   - 약함
   - 보통
   - 강함
   - 매우 강함
